## usage

```
vorbispak [cmd]
    cmd:
        help                            -- show help
        cut [option] [pak-file]         -- cut packed file to ogg viorbis files
            -o [outpath]            -- output ogg file path defaults cwd
            -p [prefix-name]        -- output ogg file name prefix
            -i [index-file]         -- index file to input
        merge [option] [ogg-files] ..   -- [WIP] merge ogg vorbis files to packed file
            -o [outpath]            -- output pak file path defaults cwd
            -n [name]               -- output pak file name
            -i [index-file]         -- index file to output
```
